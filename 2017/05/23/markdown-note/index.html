Markdown语法小记

发表于 2017-05-23 | 分类于 笔记
学习一种新的书写方式。

老郭相声中里有个段子是讲，于老师看书时桌子上会放着两本书，一本是字典，另一本是如何查字典。终于，我准备写一些技术类的博文，那么，首先要做的事情就是学习如何写博文。万幸，用键盘打字这一技能我已经掌握了，接下来准备了解一下Markdown这种写作语法。为什么我们要学习 Markdown 的三个理由给出了学习Markdown进行写作的三个理由:

  • 排版简洁明了
  • 不易产生错误
  • 不用局限于浏览器

看过之后感觉还是比较简洁易懂的，注意以下几点即可：

  • 英文模式输入
  • #和* 后面要有space
  • 加入换行避免前后之间的影响
  • 多多练习熟悉
